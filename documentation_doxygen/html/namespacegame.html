Detailed Description

Fichier qui contient la classe Game, qui gère le jeu.

Description

Classe principale du jeu, qui gère les entités, les événements, l'affichage, la boucle de jeu, etc. Contient également la classe Camera, qui gère la caméra, ainsi que la classe GameEvents, qui gère les événements du jeu. Fait appel à la classe Profiler, qui permet de profiler le jeu pour optimiser les performances. Utilise les modules que l'on a développé dans game_engine, game_objects et UI.
