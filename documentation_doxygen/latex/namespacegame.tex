\doxysubsection{Detailed Description}
Fichier qui contient la classe \doxylink{classgame_1_1_game}{Game}, qui gère le jeu. 

\hypertarget{index_description_main}{}\doxysubsection{\texorpdfstring{Description}{Description}}\label{index_description_main}
Classe principale du jeu, qui gère les entités, les événements, l\textquotesingle{}affichage, la boucle de jeu, etc. Contient également la classe \doxylink{classgame_1_1_camera}{Camera}, qui gère la caméra, ainsi que la classe \doxylink{classgame_1_1_game_events}{Game\+Events}, qui gère les événements du jeu. Fait appel à la classe Profiler, qui permet de profiler le jeu pour optimiser les performances. Utilise les modules que l\textquotesingle{}on a développé dans \doxylink{namespacegame__engine}{game\+\_\+engine}, game\+\_\+objects et \doxylink{namespace_u_i}{UI}.
