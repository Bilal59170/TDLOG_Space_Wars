\doxysubsection{Detailed Description}
Classe principale du jeu. 

Attributs \+:
\begin{DoxyItemize}
\item endgame \+: booléen qui indique si la partie est terminée
\item map \+: instance de la classe Map
\item player \+: instance de la classe Ship
\item asteroids \+: liste d\textquotesingle{}instances de la classe Asteroid
\item ennemies \+: liste d\textquotesingle{}instances de la classe Ennemy
\item entities \+: liste d\textquotesingle{}instances de la classe Entity
\item batch \+: batch d\textquotesingle{}objets
\item time \+: ticks de jeu
\item window \+: fenêtre pyglet
\end{DoxyItemize}

Méthodes \+:
\begin{DoxyItemize}
\item remove \+: supprime un objet de la liste d\textquotesingle{}entités
\item display \+: affiche la fenêtre de jeu
\item update \+: met à jour les entités
\item run \+: lance la boucle de jeu
\end{DoxyItemize}

Méthodes de gestion des événements \+:
\begin{DoxyItemize}
\item on\+\_\+close \+: ferme la fenêtre de jeu
\item on\+\_\+key\+\_\+press \+: gère les événements liés aux touches du clavier 
\end{DoxyItemize}
